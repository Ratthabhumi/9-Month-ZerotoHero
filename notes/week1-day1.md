# Week 1 Day 1 — Developer Environment & Git Foundations

**Date:** 2026-07-10
**Theme:** Developer Environment & Git Foundations

---

## ✅ What I Completed Today

1. **Setup Environment:** ติดตั้งและใช้งาน VS Code, Git, PowerShell เบื้องต้น
2. **Linux Commands:** เข้าใจคำสั่งพื้นฐาน (ls, cd, mkdir, touch, rm, pwd, clear) และความแตกต่างเมื่อใช้บน PowerShell/Linux
3. **Git Basics:** 
   - `git init`: สร้าง Repository ใหม่
   - `git add .`: นำไฟล์เข้า Staging Area เตรียมบันทึก
   - `git commit -m "..."`: บันทึกประวัติพร้อมข้อความอธิบาย
   - `git push`: ส่งโค้ดขึ้น Remote Repository (GitHub)
4. **.gitignore:** เรียนรู้วิธีละเว้นไฟล์ที่ไม่ต้องการให้ Git ติดตาม (เช่น node_modules, .env, venv)

## 💡 Key Takeaway
Git เป็นระบบ Version Control ที่สำคัญมาก ทำให้เราสามารถย้อนกลับไปดูโค้ดเก่าๆ ได้ และทำงานร่วมกับคนอื่นได้ง่ายขึ้น คำสั่งพื้นฐานอย่าง add, commit, push ต้องใช้ให้คล่องเพราะต้องใช้ทุกวัน!
