## 💡 Key Concepts ของสัปดาห์นี้

### Version Control คืออะไร?
Version Control คือระบบที่ช่วยติดตามการเปลี่ยนแปลงของโค้ด
เหมือน "ย้อนเวลา" ได้นั่นเอง

### Git vs GitHub
- **Git** = เครื่องมือ (ติดตั้งในเครื่อง)
- **GitHub** = บริการเก็บโค้ดออนไลน์ (cloud)
